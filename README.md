JS LocalStorage Playground is a small JavaScript demo that shows how
browser localStorage works using simple user input.

It is designed for beginners who want to understand
client-side storage behavior.

---

## What This Script Demonstrates

- Setting values in localStorage
- Reading values from localStorage
- Removing specific keys
- Clearing all stored data
- Using conditions with stored keys

---

## Why This Is Useful

localStorage is commonly used for:
- Session data
- Preferences
- Feature flags
- Temporary client-side storage

Understanding how it works is important for
both frontend development and security awareness.

---

## Usage

1. Open browser DevTools
2. Paste the script into the console
3. Enter a key and value when prompted
4. Observe changes in localStorage

---

## Important Notes

- localStorage is NOT secure storage
- Data persists until cleared
- Sensitive data should never be stored here

---

## Learning Outcome

This project helps you understand:
- Browser storage APIs
- Data persistence
- Client-side data risks
